# СУБД «Предприятие» (Курсовая работа)

Консольная программа на языке C для управления, поиска и сжатия текстовой базы данных сотрудников (`testBase2.dat`). Проект написан модульно с возможностью переиспользования структур данных.

## Основной функционал
* **Загрузка БД:** Чтение бинарного файла сотрудников (записи по 64 байта: ФИО, отдел, должность, дата рождения).
* **Цифровая сортировка (Radix Sort):** Поразрядное упорядочивание данных по дате рождения.
* **Модифицированный двоичный поиск:** Поиск по году рождения для формирования очереди.
* **Дерево оптимального поиска (ДОП):** Построение бинарного дерева по номерам отделов на основе частот (приближенный алгоритм А1) и поиск в нем.
* **Кодирование Гилберта-Мура:** Оценка вероятностей символов файла, построение почти оптимального алфавитного кода и вывод статистики сжатия.

## Структура проекта
* `list.c` / `list.h` — Универсальный двусвязный список.
* `bintree.c` / `bintree.h` — Бинарное дерево поиска и функции его обхода.
* `binsearch.c` — Реализация двоичного поиска через массив указателей `void*`.
* `radixsort.c` — Алгоритм поразрядной сортировки.
* `compress.c` — Анализ частот и генерация кодов Гилберта-Мура.
* `database.c` / `database.h` — Структура записи `Person`, фильтры и текстовое меню.
* `main.c` — Точка входа в программу.

## Компиляция и запуск
Сборка осуществляется через `gcc` и `make`:

```bash
make          # Компиляция (бинарник появится в ./build/app)
./build/app   # Запуск (требуется наличие testBase2.dat в корне)
make clean    # Очистка артефактов сборки
```
